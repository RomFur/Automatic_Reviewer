# Setup guide

## 1 Setup backend

### 1.1 Create virtual environment:
```bash
   python -m venv venv
```
### 1.2 Activate the environment:
 On windows: 
 ```bash
cd backend
venv\Scripts\activate
```
On Mac/Linux:
 ```bash
source venv/bin/activate
```

### 1.3 Install dependencies:
```bash
cd backend
pip install -r requirements.txt
